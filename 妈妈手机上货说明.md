# 妈妈手机上货说明

## 平时怎么用

1. 用妈妈自己的手机打开 `https://shop.reverser9n6h12.me/mobile-admin.html`。
2. 建议在浏览器菜单里选择“添加到主屏幕”，以后点桌面图标进入。
3. 点“新增商品”，拍商品正面、型号标签和重要细节。
4. 填商品名、分类、价格、品牌、型号和一句话介绍。
5. 有明确资料时再展开“详细介绍与参数”，不确定的参数不要猜。
6. 点“发布到网站”。看到“发布成功”后等待 1-3 分钟，再打开客户网站查看。

修改旧商品：在商品列表里点商品，修改后点“发布到网站”。

## 第一次连接（由孩子操作）

手机后台不会在源码里保存密码。第一次需要创建一个 GitHub fine-grained token，并在妈妈手机上粘贴一次：

1. 登录店铺使用的 GitHub 账号。
2. 打开 `https://github.com/settings/personal-access-tokens/new`。
3. Repository access 只选择 `Only select repositories`，再选择 `pinshang`。
4. Repository permissions 里只把 `Contents` 设为 `Read and write`，其他权限不要增加。
5. 生成 token 后，只在妈妈手机的“管理员密钥”框里粘贴并连接。
6. 连接成功后删除临时复制记录，不要通过微信转发 token。

建议设置合理的有效期。到期后网站照常展示，只是手机后台需要由孩子重新生成并连接一次。

## 注意

- 这个后台地址不要发朋友圈，也不要发给客户。
- 妈妈不需要安装 GitHub，也不需要理解代码。
- 换手机、清理浏览器数据、使用无痕模式，都会丢失本机保存的管理权限。
- 发布时需要能够连接 GitHub API；若提示网络失败，先换移动数据或 Wi-Fi 重试。
- 删除商品只会让它从商品列表消失，不会自动清理仓库里的旧图片。
