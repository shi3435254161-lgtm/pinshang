# 妈妈上货与发布说明

这个网站现在分两部分：

1. 客户看的正式网站：`https://shop.reverser9n6h12.me/`
2. 自己电脑上用的商品助手：`F:\pinshang-shop\start-admin.bat`

## 上货流程

1. 双击打开：

```text
F:\pinshang-shop\start-admin.bat
```

2. 浏览器会打开商品管理助手。
3. 点左边商品，或者点“新增商品”。
4. 填商品名、价格、品牌、型号、图片、参数、安装说明。
5. 点页面底部“保存当前商品到列表”。
6. 点右上角“保存到 data.js”。
7. 关闭商品助手窗口。
8. 双击发布：

```text
F:\pinshang-shop\publish-products.bat
```

发布成功后，客户访问的网站就会更新。

## 图片怎么放

商品图片在商品助手里上传即可，它会放到：

```text
F:\pinshang-shop\assets\products\
```

业务微信二维码建议命名为：

```text
F:\pinshang-shop\assets\wechat-qr.jpg
```

然后在商品助手的“店铺联系设置”里填写：

```text
微信二维码路径：assets/wechat-qr.jpg
业务微信号：妈妈接单用的微信号
```

## 注意

发布脚本只会上传商品数据、商品图片和微信二维码，不会上传 PDF、Python 脚本、临时文件夹等杂项。

如果妈妈不会电脑，最稳的办法是：妈妈把商品照片、价格和型号发给你，你远程用商品助手上货，然后双击发布。
