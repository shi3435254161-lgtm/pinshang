# 联塑管道 · 品尚装饰产品站

这是一个不依赖服务器和数据库的手机端产品目录，主要承接朋友圈里的产品介绍、安装咨询和电话预约。

## 当前状态

- 已录入店名、服务范围和两个联系电话。
- 已建立热水器、烟机灶具、卫浴和水电管道分类。
- 已实现搜索、分类筛选、产品独立详情链接、电话联系、询价内容复制和手机底部快捷操作。
- 产品图片暂时从现有朋友圈截图裁切，只适合作为临时素材。
- 真实型号、价格、安装包含内容、门头照片、业务微信二维码和完工案例等待补充。

## 打开网站

直接双击 `index.html` 即可预览。正式分享前，应部署到支持 HTTPS 的静态托管服务，并在本地移动、联通、电信网络及微信内测试。

如果要发布到朋友圈或自己上货改价，先看 `上货与朋友圈发布教程.md`。里面写了上线拿链接、朋友圈分享、改商品、换图片、加二维码和加真实案例的具体模板。

如果使用 GitHub Pages 上线，按 `GitHub Pages部署教程.md` 操作。

## 修改产品

日常上货建议优先使用本地商品管理助手，双击 `start-admin.bat` 打开。它可以用表单维护商品、上传商品图、保存 `data.js` 并生成上传文件夹。

如果只有手机，可以打开 `https://shop.reverser9n6h12.me/mobile-admin.html`。第一次由懂电脑的家人配置一个仅限 `pinshang` 仓库、只有 Contents 读写权限的 GitHub fine-grained token；配置完成后，手机端可直接新增、修改、拍照上传和发布商品。管理密钥只保存在那台手机的浏览器里，不能写进源码、聊天记录或公开页面。

所有产品资料最终都会集中保存到 `data.js`。每个产品包含：

- `name`：对外产品名称
- `brand`：品牌
- `model`：准确型号
- `image`：图片路径
- `price`：产品价或产品加基础安装价
- `priceNote`：价格的简短说明
- `aliases`：客户可能会搜索的俗称，例如“马桶”“煤气灶”
- `gallery`：商品图册，支持多张图片
- `sellingPoints`：商品卖点
- `specs`：规格参数表
- `buyingGuide`：选购说明
- `installCheck`：安装前需要客户确认的内容
- `included`：基础安装默认包含项目
- `extras`：可能另收费的项目
- `faqs`：客户常问问题
- `notice`：选型或施工注意事项

没有得到确认的信息不要猜，继续保留“待确认”或暂时隐藏该产品。

## 微信二维码

业务微信二维码是负责接单的联系人二维码，不是微信收款码。建议使用专门的业务微信，并保持头像、店名、电话与网站一致。

二维码图片准备好后：

1. 放到 `assets/wechat-qr.jpg`。
2. 将 `data.js` 中的 `wechatQr` 改成 `assets/wechat-qr.jpg`。
3. 重新打开页面，联系窗口会自动显示二维码。

如果有固定业务微信号，也可以把 `data.js` 中的 `wechatId` 改成实际微信号，联系窗口会显示“复制微信号”。个人微信通常不能从普通网页稳定直接跳到聊天或加好友，除非后续使用企业微信客服、公众号、小程序等官方能力。

## 真实案例

真实案例不是好评率，而是一笔已经完成的施工记录：

- 所在村镇，只写到村级并取得客户同意
- 客户原来的问题
- 施工前照片
- 施工过程或关键细节
- 完工照片与测试结果
- 使用产品和解决方案
- 客户评价可作为额外佐证，但不要伪造

公开前遮挡人脸、门牌、电话号码、快递单和车牌。

## 基础安装包含项

它表示公开报价默认承诺包含的工作。最终口径必须由家里根据实际施工方式确认。例如：

| 品类 | 可讨论的基础包含项 | 常见另收费项 |
| --- | --- | --- |
| 热水器 | 服务范围内送货、常规挂机、水路连接、通水通电测试 | 拆旧、特殊打孔、超长管线、线路整改 |
| 烟机 | 定位固定、常规烟管连接、通电试机 | 拆旧、扩孔、止逆阀、加长烟管、高空施工 |
| 马桶 | 常规定位、密封、通水试验 | 拆旧、移位、坑距整改、地面修补、额外配件 |
| 花洒 | 常规冷热水口安装、密封、通水试验 | 拆旧、暗管整改、特殊墙面打孔、额外配件 |
| 浴室柜 | 入位固定、常规给排水连接、通水测试 | 拆旧、墙体加固、水电改位、额外配件 |

燃气连接与改造必须按照当地规定和资质要求执行，不能把不具备条件的项目写进基础安装承诺。

## 上线前检查

- 用原图替换临时裁切图片，单张建议 1200 像素左右、清晰明亮。
- 填写准确品牌、型号、价格和更新时间。
- 家里逐条确认安装包含项和另收费项。
- 上传业务微信二维码、门头照片和至少三个真实案例。
- 检查两个电话是否都能正常接听。
- 在三种运营商网络和微信内打开全部产品链接。
- 请五位熟人试用，观察是否能在一分钟内找到产品并完成咨询。
