# 上货与朋友圈发布教程

这个站本质上是一个静态产品目录，没有数据库，也没有后台。以后上货、改价、换图，主要改 `data.js`，图片放到 `assets/products/`。

你学过 Hexo 的话，可以把它理解成：

- `data.js`：商品数据，类似 Hexo 的配置和文章 front matter。
- `assets/products/`：商品图片目录。
- `index.html`：页面骨架，一般不用动。
- `styles.css`：页面样式，想改颜色、间距、字体时再动。
- `app.js`：搜索、弹窗、复制询价内容等交互逻辑，一般不用动。

## 一、朋友圈为什么现在不能直接发

你现在浏览器里打开的 `http://127.0.0.1:4173/` 是本地地址，只能你这台电脑访问。发到朋友圈之前，必须先上线，拿到一个 `https://...` 开头的公网链接。

上线以后，朋友圈发布流程是：

1. 把公网链接发到自己的微信，建议先发“文件传输助手”。
2. 在手机微信里打开这个链接。
3. 点右上角 `...`。
4. 选择“分享到朋友圈”。
5. 配一段短文，例如：

```text
宁陵县及周边乡村厨卫、水电、烟机灶具可以咨询。
热水器、马桶、花洒、浴室柜、油烟机，支持到店看货和上门安装。
点进去看产品，需要的直接微信或电话联系。
```

普通网页不能真正做“点击按钮一键发朋友圈”。微信里的自定义分享标题、缩略图、描述，通常需要公众号 JS-SDK、可信域名和签名服务。第一阶段先用手动分享，成本最低，也够练单。

## 二、零成本上线建议

第一阶段建议先用零成本静态托管练习，目标不是一步到位，而是先让亲戚、老客户、附近村镇的人能点开。

推荐顺序：

1. EdgeOne Pages / Pages Drop：腾讯系静态托管，适合先测试微信内打开情况。
2. GitHub Pages：你学过 Hexo，应该很容易上手；缺点是国内和微信内速度需要实测。
3. Cloudflare Pages / Netlify：也能免费托管静态站，但同样要测试微信内是否稳定。

上传时注意：网站根目录必须直接看到 `index.html`。也就是说要上传 `F:\pinshang-shop` 里面的这些内容，而不是把外层 `pinshang-shop` 套一层再传。

应该包含：

```text
index.html
styles.css
data.js
app.js
assets/
```

拿到公网链接后，先用你妈妈手机、你爸手机、移动/联通/电信网络各测一次。能打开、能看产品、能复制微信询价内容，再开始发朋友圈。

如果 EdgeOne Makers / Pages 访问时出现 `401 Unauthorized`，说明这个链接可能是预览链接、访问凭证过期，或者当前区域策略不允许直接访问。不要把这种链接发朋友圈。用无痕浏览器、手机流量和亲戚手机测试都能打开，才算是真正可用。

## 二点五、每次更新后重新生成上传包

改完商品、图片、二维码以后，可以在 PowerShell 里运行：

```powershell
F:\pinshang-shop\build-upload-package.ps1
```

它会刷新这个上传文件夹：

```text
F:\pinshang-shop-upload
```

也会生成一个带时间的新 zip 包：

```text
F:\pinshang-shop-deploy-日期时间.zip
```

如果平台支持拖文件夹，优先拖 `F:\pinshang-shop-upload`。如果平台只支持 zip，再传新生成的 zip。

## 三、改一个现有商品

打开：

```text
F:\pinshang-shop\data.js
```

找到类似这一段：

```js
{
  id: "water-heater-60l",
  category: "热水器",
  name: "60L 储水式电热水器",
  brand: "美的",
  model: "具体型号待确认",
  image: "assets/products/water-heater.png",
  price: "价格待录入",
  priceNote: "商品和安装费用分开确认",
  description: "适合常见家庭洗浴场景，购买前需结合人数、连续使用习惯、安装位置和线路条件选型。",
  tags: ["容量 60L", "送装可咨询"],
  aliases: ["热水器", "电热水器", "洗澡", "洗浴", "储水式"],
  included: ["基础安装内容待店内确认后公开"],
  extras: ["拆除旧机", "超长水管或电线", "墙体特殊打孔", "线路或漏保整改", "超出服务范围的配送"],
  notice: "安装前应确认墙体承重、接地、漏保与线路条件。"
}
```

常改的字段：

- `name`：客户看见的商品名。
- `brand`：品牌。
- `model`：型号，能写准确就写准确。
- `image`：商品图片路径。
- `price`：价格，比如 `¥699 起`、`到店/微信询价`、`产品价 ¥499，安装另议`。
- `priceNote`：价格补充说明。
- `description`：一句话讲清楚适合谁。
- `tags`：卡片上的小标签，2 到 4 个就够。
- `aliases`：客户可能搜的土话、俗称，比如马桶、坐厕、抽烟机、煤气灶。
- `gallery`：商品图册，可以放多张实拍图。
- `sellingPoints`：商品卖点，像电商详情页里的核心卖点。
- `specs`：规格参数，像京东淘宝里的参数表。
- `buyingGuide`：选购说明，告诉客户怎么选。
- `installCheck`：安装前确认，让客户知道要拍哪些照片、量哪些尺寸。
- `included`：这个价格默认包含什么。
- `extras`：哪些情况可能加钱。
- `faqs`：客户常问问题。
- `notice`：选型和施工提醒。

改完以后刷新网页。如果页面空白，大概率是逗号、引号、括号写错了。

## 四点五、商品资料从哪里来

不要直接复制京东、淘宝、拼多多的详情页图片和文案。更稳的资料来源是：

- 商品包装、铭牌、说明书
- 供应商给的电子画册或产品参数表
- 品牌官网公开参数
- 门店实拍图
- 你爸实际安装经验
- 客户安装现场照片，公开前必须遮挡隐私

如果资料不确定，就写“以实物为准”“需现场确认”。乡镇生意讲清楚，比写得夸张更重要。

`id` 很重要，它决定这个商品的独立链接。只能用英文、数字和横杠，不要用中文、空格。

比如网站上线地址是：

```text
https://你的域名/
```

这个商品的独立链接就是：

```text
https://你的域名/?product=liansu-ppr-20
```

以后朋友圈可以专门发某一个商品，不一定只发首页。

## 五、增加新分类

分类在 `data.js` 顶部：

```js
categories: ["全部", "热水器", "烟机灶具", "卫浴", "水电管道"],
```

如果要加“净水器”，改成：

```js
categories: ["全部", "热水器", "烟机灶具", "卫浴", "水电管道", "净水器"],
```

然后新增商品时写：

```js
category: "净水器",
```

分类名字必须一模一样。

## 六、换商品图片

图片放这里：

```text
F:\pinshang-shop\assets\products\
```

建议规则：

- 文件名用英文和横杠，例如 `midea-water-heater-60l.jpg`。
- 不要用中文文件名，不要带空格。
- 单张图片建议 800 到 1200 像素宽。
- 尽量用实拍清晰图，别用朋友圈截图裁切图长期当主图。
- 图片太大可以先压缩，单张控制在 500KB 以内比较舒服。

放好图片以后，在商品里改：

```js
image: "assets/products/midea-water-heater-60l.jpg",
```

路径里的斜杠用 `/`，不要写成 Windows 的 `\`。

## 七、加业务微信二维码

二维码是接单联系人的微信二维码，不是收款码。

把二维码图片放到：

```text
F:\pinshang-shop\assets\wechat-qr.jpg
```

然后在 `data.js` 里改：

```js
wechatQr: "assets/wechat-qr.jpg"
```

如果有固定业务微信号，也可以在 `data.js` 里填：

```js
wechatId: "你的业务微信号"
```

刷新页面，点“微信咨询”，弹窗里就会显示二维码和复制微信号入口。

普通个人微信不能像电话一样从网页稳定直接跳到聊天或加好友。微信出于防骚扰和安全限制，一般不开放个人号网页直达聊天。第一阶段最稳的是“复制咨询内容 + 长按识别二维码 + 复制微信号”。如果后续愿意升级，可以考虑企业微信客服、公众号客服或小程序客服链接。

## 九、每次改完怎么检查

最少做这几步：

1. 刷新首页，看商品是否还正常显示。
2. 搜索一个关键词，比如“马桶”“烟机”“水管”。
3. 点商品详情，看图片、价格、安装包含项是否正确。
4. 点微信询价，看复制出来的文字是否正常。
5. 如果已经上线，重新上传文件后用手机微信打开公网链接。

如果网页突然空白，先检查 `data.js`：

- 字符串必须用英文引号：`"这样"`。
- 数组中每一项之间要有英文逗号。
- 商品对象之间要有英文逗号。
- 最后整体结构必须还是 `window.SHOP_DATA = { ... };`。
- VS Code 右下角编码保持 `UTF-8`。

## 十、朋友圈文案模板

首页通用版：

```text
宁陵县及周边乡村，厨卫、水电、烟机灶具可以咨询。
热水器、马桶、花洒、浴室柜、油烟机，支持到店看货和上门安装。
点进去先看产品，有需要直接微信或电话联系。
```

单品促单版：

```text
最近家里要换【热水器/马桶/油烟机】的可以看看。
我们店在村里，宁陵县及周边乡村可咨询安装。
价格、安装包含什么、哪些情况另收费，都可以先微信问清楚。
```

案例信任版：

```text
今天整理了几个附近村里的安装案例。
买厨卫电器不只是看价格，关键是型号选对、安装稳、后续能找得到人。
需要热水器、马桶、花洒、烟机灶具、水电维修的，可以点进去看看。
```

别一条朋友圈塞太多商品。朋友圈更适合“一次主推一个品类”，比如今天热水器，明天马桶，后天烟机灶具。
