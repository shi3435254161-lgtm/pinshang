# 商品管理助手使用说明

这个助手只在你自己的电脑上用，不是发给客户看的后台。客户只访问正式网站，浏览商品后点电话或微信咨询。

## 怎么打开

双击：

```text
F:\pinshang-shop\start-admin.bat
```

它会打开本地管理页面：

```text
http://127.0.0.1:5177/admin.html
```

关闭命令窗口，管理助手就停止。

## 日常上货流程

1. 打开商品管理助手。
2. 点左侧商品，或者点“新增商品”。
3. 填商品名、价格、图片、参数、卖点、安装说明、常见问题。
4. 点“保存当前商品到列表”。
5. 点右上角“保存到 data.js”。
6. 点“生成上传文件夹”。
7. 上传 `F:\pinshang-shop-upload`，或者用 git 推送到 GitHub Pages。

## 图片怎么弄

在“上传商品图片”里选择图片，助手会把图片放到：

```text
F:\pinshang-shop\assets\products\
```

并自动写入商品主图和图册路径。建议图片先压缩，单张最好不要超过 5MB。

## 微信客服怎么填

微信客服就是你妈妈负责接单的业务微信。

在“店铺联系设置”里填：

- 业务微信号：你妈妈的业务微信号
- 微信二维码路径：`assets/wechat-qr.jpg`

二维码图片仍然需要你放到：

```text
F:\pinshang-shop\assets\wechat-qr.jpg
```

客户在网站里点“微信咨询”，就能看到电话、二维码、微信号复制和咨询内容复制。

## 重要边界

这个方案是零成本静态网站，没有线上数据库后台。也就是说：你在本地助手里改完后，还要重新部署到 GitHub Pages，客户才能看到新内容。

它适合现在这个阶段：你一个人当小老板，自己上货、自己维护，客户只负责浏览和咨询。
